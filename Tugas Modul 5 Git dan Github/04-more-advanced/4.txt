1. What is the difference between git reset and git revert. When would you use one over the other?
   git reset berfungsi untuk mengembalikan history file dengan menghapus history commit yang ada
   git revert berfungsi untuk mengembalikan history file tanpa menghapus history commit yang ada
2. What is the difference between git merge and git rebase. When would you use one over the other?
   git merge adalah fungsi untuk menggabung branch menjadi satu
   git rebase adalah fungsi untuk memindahkan cabang dari satu lokasi ke lokasi lain
3. What is the difference between git stash pop and git stash apply. When would you use one over the other?
   git stash pop membuang simpanan setelah menerapkannya
   git stash apply tetap menyimpan simpanan di daftar simpanan untuk kemungkinan nanti digunakan kembali meskipun telah menerapkannya 
4. What kinds of things can you do in interactive mode when rebasing?
   bisa digunakan untuk editing (mengedit), deleting (menghapus), dan squashing (meringkas)
